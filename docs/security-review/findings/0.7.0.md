for 0.8.0.

| 3 | Passphrase in cleartext over the socket | **Fixed (0.7.0)** — client-side key derivation |

| 8 | No connection limit on the daemon (DoS) | **Fixed (0.6.0)** — configurable ceiling + read timeout |
| 9 | No SCA / `cargo audit` in CI | **Fixed (0.7.0)** — `cargo audit` gate |
| 10 | Transitive advisories via `ratatui` | **Fixed (0.7.0)** — `ratatui` 0.30, audit clean |
